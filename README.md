# bcctheme

This package provides a ggplot2 theme and colour palettes for for use by Birmingham City Council public health data analysts.

## Installation

The package can be installed from GitHub using devtools:

```{r}
# install.packages("devtools")
devtools::install_github("BCC-PHM/bcctheme")
```

## Usage

Documentation for the package can be found [here]("https://bcc-phm.github.io/bcctheme/").
